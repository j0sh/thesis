An interesting way of detecting edges, particularly in noisy images.

Method:
  1. Take the 2-D DCT of each 8x8 block in the image.
  2. For each block, select the largest AC value and store in a matrix mat.
  3. Run a morphological dilation (with a disk of radius 2) over mat.
  4. Find a threshold for mat, apply threshold to mat.
  5. Use thresholded image to mask the 8x8 blocks in the original image.
  6. Run inverse DCT over masked blocks.

This works because although noise creates a lot of high-frequency components, it is not very intense compared to important parts of the image (impulse noise excepted). Therefore, using the largest AC value as a heuristic to filter high-frequency noise works well. Limited testing shows promising results with Poission and additive Gaussian noise. Unsuprisingly, efficacy is reduced as the Gaussian variance increases (although it is mostly unaffected by the additive mean). Impulse noise is also manageable given a density around 5% or lower. Images with a cluttered background do not work well.

Results are favorable against Sobel and Difference-of-Gaussian edge detection. While Sobel and DoG give accurate contours with clean images, they do not give blobs. (Blob sizes can be adjusted via the structuring element, or the DCT.) With noise, both Sobel and DoG exhibit poor performance.
