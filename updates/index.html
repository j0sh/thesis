Proposal approved. Abstract:

A new type of background subtraction algorithm is proposed. A patch-based approximate nearest-neighbor (ANN) image correspon- dence algorithm can be used to handle background segmentation in video, even with translational camera motion. A single-image fore- ground detection algorithm, also based off the same ANN data struc- ture, can be used for immediate background initialization, and to avoid mis-labeling stationary foreground objects.
